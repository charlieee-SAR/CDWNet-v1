# Data Access

The OS-CDW dataset is distributed through one external Baidu Netdisk link.

- Baidu Netdisk URL: `https://pan.baidu.com/s/14IUaBTI2AjPX8H_E7RexDQ?pwd=exxw`

The downloaded package should contain the following folders:

- `train-data`: the tiled dataset used for training.
- `pretrained`: the pretrained backbone weights required by CDWNet.
